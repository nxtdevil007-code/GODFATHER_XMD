A powerful, feature-rich WhatsApp Multi-Device bot built with Baileys.

## ✨ Features

### 🛠️ Tools
- Calculator, Weather, Dictionary, Translator
- Text-to-Speech, QR Code, Screenshot
- URL Shortener, IP Lookup, Carbon Code
- Timer/Reminder, Base64, Binary converter

### 🤖 AI
- ChatGPT / OpenAI integration
- AI Image Generation (DALL-E style)
- Gemini AI
- Code Assistant
- Text Summarizer

### 🔍 Search
- Google, Wikipedia, YouTube
- GitHub, Pinterest, Image search
- Movie/Show search, News

### 📥 Downloads
- YouTube Audio/Video
- Instagram, TikTok, Facebook
- Twitter, Generic URL fetcher

### 🎨 Stickers
- Image/Video to Sticker
- Circle, Rounded, Crop stickers
- Sticker to Image, Text Sticker
- URL to Sticker, GIF Sticker
- Steal/Take sticker with custom pack

### 🔄 Converters
- Audio to Voice Note & vice versa
- Video to Audio
- Media to Document
- Image to URL

### 🎮 Fun
- 8Ball, Rate, Ship
- Truth or Dare, Jokes, Facts, Quotes
- Coin Flip, Dice Roll, RPS
- TicTacToe, Slot Machine
- Roast, Compliment, Choose
- Would You Rather
- Action commands (slap, hug, pat, etc.)

### 👥 Group Management
- Kick, Add, Promote, Demote
- Mute/Unmute, Lock/Unlock
- Tag All, Hidden Tag
- Group Info, Invite Link
- Warn System, Polls
- Set Group Name/Description/Picture
- Admin list

### 🎌 Anime
- Anime/Manga/Character search
- Anime Quotes
- Waifu, Neko, Shinobu images
- 20+ anime image categories

### 👑 Owner Features
- Eval, Shell command
- Broadcast to all groups
- Block/Unblock users
- Join/Leave groups
- Public/Private mode
- Anti-Link, Anti-Delete, Anti-Call
- Auto-Read, Auto-Status View
- Set Profile Picture/Bio

### 🛡️ Security
- Anti-Spam protection
- Anti-Link (auto-kick)
- Anti-Delete (log deleted messages)
- Anti-Call (auto-reject)
- Warning system (3 strikes = kick)

## 🚀 Installation

### Prerequisites
- Node.js 18+
- FFmpeg (for media processing)
- Git

### Setup

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/GODFATHER-XMD.git
cd GODFATHER-XMD

# Install dependencies
npm install

# Edit config
nano config.js
# Add your owner number and API keys

# Start the bot
npm start
